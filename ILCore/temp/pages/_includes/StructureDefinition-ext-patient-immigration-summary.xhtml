Summary

Complex Extension: Extension: Immigration information for patient - for the purpose of this extension - immigration is an international movement of a person to a destination country in order to settle as permanent resident

  • origin: CodeableConcept: Country code corresponding to origin country of patient.
  • destination: CodeableConcept: Country code corresponding to destination country of patient. When missing, destination country is assumed to be Israel
  • date: date: Date of immigration.
  • status: CodeableConcept: Immigration status according to LOINC answer-list LL5005-5 [http://loinc.org/LL5005-5]
